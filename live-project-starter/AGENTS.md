# AGENTS.md

This file gives AI coding agents project-specific instructions.

## Project Goal

Build a small data analysis dashboard from a Kaggle dataset for students learning AI coding tools.

## Audience

Novice programming students.

## Constraints

- Keep the implementation simple enough to explain in class.
- Prefer Python, pandas, Streamlit, and Plotly unless the instructor asks for something else.
- Keep dependencies minimal and list them in `requirements.txt`.
- Prefer readable code over clever abstractions.
- Make the dashboard readable on laptop screens.
- Use accessible headings, labels, color contrast, and chart titles.
- Do not include Kaggle API keys, account credentials, or secrets.
- Do not commit large downloaded datasets unless the instructor explicitly asks for a small sample.

## Required Content

- Dashboard title and short dataset summary.
- At least three charts that answer clear questions about the data.
- A small filter or selector.
- A short notes section explaining limitations or missing data.

## Workflow

1. Inspect before editing.
2. Make one small, reviewable edit at a time.
3. Run the dashboard locally and check the charts.
4. Summarize what changed and how it was checked.

## Design Direction

Use a clean dashboard style. Favor readable text, clear chart labels, enough spacing, and a layout that helps students understand the data quickly.
