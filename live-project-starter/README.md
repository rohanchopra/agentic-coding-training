# Live Project Starter

Use these files to guide a workshop live project where students build a small data analysis dashboard from a Kaggle dataset.

The completed dashboard should be created outside this starter folder, usually in `student-dashboard/`.

## Suggested Workflow

1. Read `docs/live-project.md` for the project brief.
2. Choose a beginner-friendly Kaggle dataset with at least one CSV file.
3. Download the dataset yourself and place the CSV in `student-dashboard/data/`.
4. Use the prompts in `prompts.md` one step at a time.
5. Review each diff before asking for another change.
6. Run the dashboard locally with Streamlit and check the charts in a browser.

## Safety Notes

- Do not commit Kaggle API keys, account credentials, or local auth files.
- Do not commit large downloaded datasets.
- Ask the AI tool to inspect the data before building charts.
- Keep the first version small enough to explain in class.

## Recommended Stack

- Python
- pandas
- Streamlit
- Plotly
